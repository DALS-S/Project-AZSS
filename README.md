# AZSS.GG
 게이밍 유저들을 위한 커뮤니티 웹 프로젝트

## 🖥 프로젝트 소개
AZSS.GG는 게임 속 경험과 실력을 공유 할 수 있는 공간 입니다.
좋은 아이템을 획득하거나, 엄청난 실력으로 게임 속에서 활약한 순간들을
기록으로 남겨 다른 사람들과 공유 할 수 있도록 제작 된 사이트입니다.
